Arguments

Kepler provides various commandline arguments that can alter the behavior of the interpreter.

How to Use

Arguments are very easy to use, and are passed when you execute the interpreter.

For example, this would launch Kepler and load a specific file from another location. kepler.exe --file "C:\Users\John_Doe\Desktop\test.kep"

Commandline Arguments

This table shows all available arguments, what they do, and any required values they might need
Name Description Req. Value
--file The pathlike directory/filename of the kepler file. Any filename or location.
--headless Run in a "headless" mode.
--help Show the list of arguments.
--version Display the currently installed Kepler version.
--debug Enable debug logging. No value, or "verbose" for verbose debug.
--langserver Disable normal functionality and provide language server features.
