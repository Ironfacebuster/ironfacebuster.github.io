Introduction

What is Kepler?

Kepler is an interpreted language, with a special syntax: it's designed to be like talking in English!

Kepler can do many things, and that list is always growing!
Here's an example of a Hello World program: hello_world.kep 1 print "Hello, world!" output Hello, world! It really is that easy!

Getting Sophisticated

Kepler can do more than just print a string!
For example, you can define variables, and use them in your code: hello_var.kep 1 name is "I'm a variable!" 2 3 print name output I'm a variable! As they always say, Function follows Variable.

In essence, functions are blocks of code that can be run multiple times, with different parameters.

In order to write a function, you need to define the function and its parameters in the code Header.
This entails telling Kepler the Type your function returns (even if it doesn't return one) and the name and type of its parameters
For example, you can define a function that takes a string and prints it: funct_print.kep 1 start Header 2
!-- Define the function --! 3
printer returns Int 4
!-- Define the "input" parameter --! 5
printer uses Int as input 6 end Header 7 8 start printer 9
print input 10 end printer 11 12 !-- run (call) the function --! 13 call printer with "This is a function that prints a string!" as input output This is a function that prints a string!
If you thought to yourself, "what's a string?" This next section is for you!

Kepler Types

Kepler only has a couple of types of data:
  • Boolean
  • Float
  • Int
  • uInt
  • String
Types can be changed to others types via "explicit casting".
explicit_casting.kep 1 int is 1 2 3 print int cast Float output 1.0 You can also use "implicit casting", or implied casting, to change the type of a variable automatically.
This is done when two values of different types are compared to each other.
implicit_casting.kep 1 int is 100.0 2 3 !-- add an Int to a Float --! 4 print int + 1 output 101.0 Implicit casting can lead to errors, so always make sure your data is sound before using it!
