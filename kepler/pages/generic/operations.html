Operations

Kepler provides a number of operations that can be performed to manipulate values.
All operations, unless explicitly stated, will return a type of the first parameter of the operation.

Addition

Add two values together.

The operator for addition is +.
The following is an example of how to use the addition operator: add.kep 1 print 1 + 1 output 2

Compatibility Table

Some operations may not be available on all types, or will operate differently depending on the type.
Type Compatible
Boolean No, unless casted to a compatible type.
Float Yes.
Int Yes.
uInt Yes.
String Yes, however, doing so will result in a new string with the content appended or prepended.

Modulo

Divide a number and return the remainder.
Modulo always returns a Float rather than the same type as its inputs.

The operator for modulu is %.
The following is an example of how to use the modulo operator: modulo.kep 1 print 10 % 5 output 0.0
