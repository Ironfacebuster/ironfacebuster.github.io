Logical Operators

Sometimes you need to combine two or more conditions. This is where logical operators come in.

And

Return True if both statements are True.

The following is an example of how to use the "and" operator: and.kep 1 print True and False output False

Or

Return True if either statements are True.

The following is an example of how to use the "or" operator: or.kep 1 print False or True output True

Not

Invert the result of another operator.

The following is an example of how to use the "not" operator: not.kep 1 print not True and True output False
