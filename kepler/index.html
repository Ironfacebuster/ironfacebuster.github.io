What is Kepler?

Kepler is a brand new language, designed to be easy-to-use and quick to learn

Features

  • English-like syntax

    Write code like you're speaking English

  • Easy-to-import libraries

    Expand Kepler's capabilities with libraries, from input to objects

  • Verbose errors and stack traces

  • Simple data types, with implicit and explicit casting

    A list of data types, as well as castability, is available in the docs

  • Examples

    Example files with varying complexity are included in the installer
